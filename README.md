#  Python Bank System

A simple banking system developed in Python that allows users to create accounts and perform basic banking operations through the command line.

## Features
- Register a new account
- Login securely
- Check account balance
- Deposit money
- Withdraw money
- Transfer money to another user
- Change account password
- Logout

## Technologies
- Python
- Functions
- Dictionaries
- Loops
- Conditional Statements

## How to Run
1. Download or clone the project.
2. Open the project in VS Code.
3. Run the Python file.
